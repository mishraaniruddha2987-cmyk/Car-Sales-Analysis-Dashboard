# Car-Sales-Analysis-Dashboard
Car Sales Dashboard – Power BI Project
Overview

This Power BI project presents an interactive Car Sales Dashboard that analyzes sales performance, revenue, units sold, brand market share, and model trends across major automobile brands.

The dashboard helps users quickly understand:

Total revenue generated
Number of units sold
Total available variants
Revenue contribution by brand
Brand market value comparison
Top-performing car models
Sales trends over time
Dashboard Features
Key Metrics
Total Revenue: 62M
Units Sold: 4,860
Total Variants: 16
Visualizations Included
Brand-wise revenue distribution (Pie Chart)
Brand market value comparison (Bar Chart)
Top 5 best-selling car models (Horizontal Bar Chart)
Year-wise sales trend (Line Chart)
Brand filters with logos for easy navigation
